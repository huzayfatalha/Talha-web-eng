================================================================================
                                STEP 0: SETUP
================================================================================

Do this BEFORE testing anything else:

1. START XAMPP:
   - Open XAMPP Control Panel
   - Click "Start" next to Apache
   - Click "Start" next to MySQL
   - Wait for green checkmarks (✓ Running)
   - Both should show as running

2. CREATE DATABASE:
   - Go to: http://localhost/phpmyadmin/
   - Click "SQL" tab at top
   - Copy and paste this code:

   CREATE DATABASE IF NOT EXISTS crm_project;
   USE crm_project;

   CREATE TABLE customers (
       id INT AUTO_INCREMENT PRIMARY KEY,
       name VARCHAR(100) NOT NULL,
       email VARCHAR(100) NOT NULL,
       phone VARCHAR(20),
       created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
   );

   CREATE TABLE admins (
       id INT AUTO_INCREMENT PRIMARY KEY,
       username VARCHAR(50) NOT NULL UNIQUE,
       password VARCHAR(255) NOT NULL
   );

   INSERT INTO admins (username, password) VALUES ('admin', '12345');

   - Click "Go" button
   - You should see success message
   - Database "crm_project" should appear in left sidebar

3. VERIFY DATABASE CREATED:
   - You should see "crm_project" database in phpMyAdmin left panel
   - Click on it to expand
   - You should see "customers" and "admins" tables

✅ SETUP COMPLETE - Now start testing!
